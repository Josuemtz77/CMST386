How to implement LESS

L.E.S.S

LESS is an acronym that stands for Leaner Style Sheets. It is a "backwards-compitable language extension for CSS". It could also be viewed as the JavaScript styling language. As the function are much like JavaScript, LESS is able to do math, change colors, combine styling to different classes and id's.

  • Mixins: ("Mixing In") is mixing a rule-set into another rule-set. When using something like a class or id, and you have another class styling that you want to have the same styling as the previous class you simply need to drop the class name into the next one as so:

  • Variables: Allows you to control each individual elements styling.

  • Functions: With LESS developers are able to utilize functions that allow different results depending on the functions conditions. Functions such as, if, boolean, strings, math, and multiple of different ones. What the main for purpose of all these functions is to change the initial content(s) (input) into new content based off a varied of conditions.

  • Operators: Are your mathmatical outouts, capable of operating with addition, subtraction, multiplication, and division. All while being capable of accounting for conversions and order of operations. Can operate on any number, color, or variable.

  • Nested Variables: Is the ability of adding styling to a single element with multiple classes to add different styling.
